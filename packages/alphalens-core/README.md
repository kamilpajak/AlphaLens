# alphalens-core

Research pipeline package: backtest / gates / screeners / thematic / overlays /
diagnostics / paper_trade / attribution / watchdog. Pure science, no web layer.

Currently sourced from the legacy `alphalens/` top-level tree at the repo root.
F1+ of the Django migration will repackage it as a pip-installable workspace
member with a published API surface that `alphalens-django` consumes.
