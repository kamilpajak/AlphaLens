## Gate 7: Survivorship / PIT

**N/A at the screener level.** Survivorship is addressed at universe
construction via the PIT union snapshot (Phase 2.5 design doc §3) —
the screener never sees ticks outside their PIT-eligible window. The
screener-internal feature `features_as_of(ticker, asof)` is itself PIT-safe
(filing_date ≤ asof). No additional cohort-split / delisting-bias check
is meaningful when the universe is already PIT-reconstructed.

## Verdict

Layer 2d was originally CLOSED 2026-04-24 on the IS→OOS Carhart α
collapse pattern (IS t=2.14 → OOS t=0.68). This re-run uses the parquet
migration as the data source and re-affirms the verdict under uniform
pipeline coverage.
