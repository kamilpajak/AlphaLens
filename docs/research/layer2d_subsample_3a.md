# Layer 2d Experiment 3a — IS subsample stability (2026-04-28)

**RESEARCH ONLY.** Direct response to Perplexity peer review of the
2026-04-28 null-distribution writeup. Splits the IS window
(2011-01..2022-12) into two roughly equal halves and re-runs both V0_count
baseline and the random-subset null on each. Goal: separate
"set-distributional artifact" from "regime-pooling artifact".

Per Perplexity prediction:
- Period A >> Period B in IS → regime attenuation already in IS
- Both equal and high in IS → persistent selection bias

This experiment falsifies a third possibility neither of us pre-registered:
the full-IS V0 result is itself partially generated by joint-window
factor-loading instability — neither subperiod alone shows the headline α.

## Setup

- Subsamples: IS_2011_2016 (N=299), IS_2017_2022 (N=301), full IS_2011_2022 (N=603)
- Top-N=15, weekly stride=5, holding=60d signal-only, SPY benchmark
- K=500 random-subset trials per subperiod (bumped from K=100 per Perplexity #8 feedback on percentile-precision)
- Same parquet cluster cache, same PIT union, same Carhart-4F (HAC) factor data

## Critical findings

### 1. Full-IS V0 result was inflated by period-pooling, not signal magnitude

Subsample α (44.1% and 42.9%) is barely different — but the full IS reports 103.5%, **2.4× higher** than the unweighted subsample mean (43.5%). Full IS R² (0.005) is **4-10× lower** than each subperiod alone (0.022, 0.054). This is not just a sample-size t-stat boost; the **point estimate** of α inflates when the two windows are pooled.

Mechanism: the relationship between portfolio returns and the four Carhart factors changes between 2011-2016 and 2017-2022. When fit jointly, the factor loadings are averaged, residual variance increases, intercept (α) absorbs the unmodeled period-shift. Per-subperiod fits absorb that shift via cleaner factor loadings → R² rises from 0.005 to 0.022/0.054, α drops from 103% to ~44%.

**The headline 103%/y IS finding was a methodological artifact of joint-window estimation.** A correctly-run analysis would report the per-subperiod number (~44%) or use rolling/time-varying factor loadings.

### 2. Per-subperiod, neither V0 nor random subsets pass nominal significance

| Period | V0 t-stat | Null median t | Null p95 t | Passes t > 2.0? |
|---|---:|---:|---:|:---:|
| 2011-2016 | 1.63 | 1.11 | 1.19 | No (V0), No (null) |
| 2017-2022 | 1.60 | 1.54 | 1.90 | No (V0), No (null) |
| Full IS pooled | 2.14 | 2.34 | 2.50 | Yes (artifact) |

Within each subperiod alone, **the screener has never passed nominal significance**. The IS Bonferroni gate (t>2.24, n=2) was always failed by V0 even at full-IS pooled inflation. Subsample-level it fails far more decisively.

### 3. Ranking quality changes by subperiod

- 2011-2016: V0 t=1.63 at **100th percentile** of null. Ranking by `insider_count` was strictly better than random selection from the cluster-positive pool. Null α/null t are tightly clustered (std 1.5pp / 0.05) — a small, stable set effect that V0 ranking improved on.
- 2017-2022: V0 t=1.60 at **62nd percentile**. Ranking added marginal value above random. Null distribution is wider (std 6.6pp / 0.23) — more dispersion in possible portfolios.
- Full IS pooled: V0 at **3rd percentile** — ranking statistically worse than random. This conclusion was itself a pooling artifact.

The earlier writeup ("ranking added zero value, statistically worse than random") **needs to be retracted** — it was true only at the pooled level, where the comparison is contaminated.

### 4. Set-membership effect: real but weaker than thought

Per-subperiod random-subset null medians:
- 2011-2016: α=34.6%/y, t=1.11
- 2017-2022: α=39.6%/y, t=1.54
- 2023-2026 (OOS): α=13.1%/y, t=0.40

Cluster-positive subset membership did deliver positive Carhart-residual returns within each IS subperiod, but at half the magnitude originally reported (~35-40%/y vs the headline 79%) and at t-stats below 1.6 (not 2.34). OOS, that effect attenuates further to 13%/y at t=0.40.

This is still 2-7× the literature baseline (Lakonishok-Lee 2001 ~6-15%/y small-cap). So the IS effect *exists* but is harder to attribute. Plausible mechanisms not yet ruled out:

- Cluster-positive selection may correlate with unmodeled value/distress characteristics (Perplexity #2b)
- PIT universe construction artifact (Perplexity #2c)
- Small-cap regime alignment 2011-2022 that attenuates 2023-2026 (the 35→13% drop is consistent)

## Revised conclusions

1. **Layer 2d KILL verdict stands** — strengthened, not weakened. None of the per-subperiod numbers pass significance. The full-IS t=2.14 figure was inflated by period-pooling.

2. **The "ranking-invariant overfit" claim from the prior writeup needs revision.** Per-subperiod, ranking does add value (especially 2011-2016 where V0 sat at 100th percentile of null). The earlier "V0 worse than random" conclusion was an artifact of pooled-sample joint regression, not a property of the ranking scheme.

3. **The diagnostic flag (IS α > 50% AND R² < 0.01) is still useful but needs reformulation.** It triggered because of pooling artifact, not real high α. A better flag would be: **R² jumps when split into subperiods**, indicating factor-loading instability that joint estimation hides. Specifically:
   - If `R²_full < 0.5 × min(R²_subperiod)`, suspect joint-window pooling artifact
   - In our case: R²_full=0.005, R²_2011-2016=0.022, R²_2017-2022=0.054 → ratio is 0.005 / 0.022 = 0.23 < 0.5 ✓ (would fire)

4. **Set-effect is real but weak.** Both IS subperiods show null median α 35-40%/y at t ~1.1-1.5. Not a "non-existent signal", but well below significance. OOS attenuation from ~37% mean IS to 13% (~2.8× attenuation) is moderate, consistent with regime shift / efficiency improvement rather than the headline 6× attenuation that pooled-IS suggested.

## Methodological lesson (sharper)

> When IS Carhart R² is **substantially lower** than per-subperiod R², joint-window regression is being contaminated by factor-loading instability. The reported α is inflated relative to any subperiod-level estimate. Always check per-subperiod regressions before trusting an IS pooled headline. **R² instability is the diagnostic, not low R² alone.**
